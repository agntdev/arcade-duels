# Arcade Duels

Telegram bot with casual reaction/tap mini-games offering solo play and real-time two-player duels, optional persistent profiles, leaderboards, and group notifications.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
